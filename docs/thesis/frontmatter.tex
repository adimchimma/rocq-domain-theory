\chapter*{Abstract}
\addcontentsline{toc}{chapter}{Abstract}

This thesis develops modern, machine-verified foundations for
programming language semantics using \Rocq{} (formerly Coq).
We modernize and substantially extend the 2009 domain theory library
of Benton, Kennedy, and Varming~\cite{BKV2009}, formalizing
$\omega$-complete partial orders ($\omega$-CPOs), Scott-continuous
functions, the Kleene fixed-point theorem, and a cartesian closed
category structure for function spaces.
Building on this foundation, we formalize CPO-enriched categories,
locally continuous functors, enriched natural transformations, and a
recursive domain equation framework.
As validation, we implement PCF$_v$ ---a simply typed lambda calculus
with general recursion---and prove both soundness and adequacy of its
denotational semantics with respect to a big-step operational semantics.
Finally, we build some extensions toward quantum programming language
semantics via quantum CPOs based largely on the work of 
Kornell, Lindenhovius, and Mislove~\cite{KLM2024}.
The entire development comprises over 15{,}000 lines of verified
\Rocq{} code using the Hierarchy Builder framework.

\vspace{1em}
\noindent\textbf{Keywords:} domain theory, denotational semantics,
proof assistant, Rocq, enriched categories, PCF, adequacy,
quantum CPOs
