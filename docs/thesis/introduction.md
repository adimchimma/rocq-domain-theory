The Big Picture 

Imagine you're writing a program for a quantum computer—a fundamentally different type of computer that could solve certain problems much faster than regular computers. How do you know your program will work correctly? Traditional testing isn't enough for quantum systems because they behave probabilistically and can be in multiple states at once. 

This project aims to modernize and extend mathematical tools that can verify programs are correct, creating a foundation that could eventually be used for quantum programming languages. Think of it as updating the mathematical infrastructure needed before we can build verified quantum programs. 

The Approach 

I'll use a tool called Rocq (formerly known as Coq), which is like a very strict mathematician that checks every step of your reasoning. The project will modernize a 2009 library for domain theory and denotational semantics, then explore how these tools can be extended for quantum computation.
